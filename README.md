# Анализ рынка недвижимости СПб и ЛенОбласти (SQL)

## Бизнес-задача
Проанализировать исторические данные объявлений (2015–2018 гг.) для оптимизации маркетинговой стратегии агентства недвижимости: выявить самые ликвидные сегменты жилья и сезонные пики[cite: 1, 2].

## Стек технологий
* **БД:** PostgreSQL.
* **Инструменты:** CTE (`WITH`), Оконные функции (`SUM OVER`), фильтрация выбросов через перцентили (`PERCENTILE_DISC`)[cite: 1].

## Главные инсайты

* **Скорость продаж:** В СПб быстрее продаются дорогие квартиры (рынок диктует цена). В Ленобласти — наоборот: дешевое жилье продается долго, а компактные квартиры улетают первыми[cite: 2]. 
* **Сезонность:** Пик активности продавцов - осень и февраль[cite: 2]. Покупатели активнее всего в сентябре-октябре и январе[cite: 2]. Реальные сделки закрываются с лагом в 1-2 месяца после публикации[cite: 2].
* **Стабильность:** Сезонные колебания вообще не влияют на среднюю стоимость кв. метра (держится 100–108 тыс. руб.) и среднюю площадь (58–62 кв. м)[cite: 2].

## Топ-3 рекомендации
1. **Август-сентябрь и февраль** - идеальное время для агрессивного запуска рекламы по привлечению продавцов[cite: 2].
2. В Ленобласти делать ставку на оборот **небольших квартир** (58-62 кв. м) - они самые ходовые[cite: 2].
3. В Санкт-Петербурге ключевое внимание уделять **правильной оценке** крупных объектов, чтобы они не зависали на рынке[cite: 2].
